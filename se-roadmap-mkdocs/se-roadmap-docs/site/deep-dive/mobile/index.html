    Deep Dive — Mobile Engineer¶

    Career levels¶

    Focus: Build screens from designs, integrate REST APIs, fix UI bugs on iOS or Android.

    Swift (iOS) or Kotlin (Android) Auto Layout / Jetpack Compose basics REST API integration Git App lifecycle Basic local storage Xcode or Android Studio

    Focus: Own features from design to App Store, implement complex UI patterns and offline support.

    SwiftUI or Jetpack Compose (advanced) CoreData / Room DB Push notifications (APNs / FCM) Authentication (Sign in with Apple / Google) Performance profiling Unit testing (XCTest / JUnit) CI/CD (Fastlane)

    Focus: Architect the app, define patterns, lead platform decisions, ensure quality at scale.

    App architecture (MV*, TCA, MVVM) React Native or Flutter (cross-platform) On-device ML (CoreML / TFLite) Custom native modules Deep linking & universal links App security App Store / Play Store operations

    Focus: Lead mobile platform strategy, drive AI-native mobile experiences, set quality standards.

    Cross-platform strategy On-device LLM integration AR frameworks (ARKit / ARCore) Mobile AI platform design Performance culture Mentoring & hiring Platform-level SDK design

    Learning path¶

    1. Swift or Kotlin fundamentals (2–3 months · Free) Apple's "Develop in Swift" or Google's Android Basics in Kotlin — official, free, excellent.

    2. SwiftUI or Jetpack Compose (1–2 months · Free) Hacking with Swift (Paul Hudson) for iOS. Google's Compose pathway for Android.

    3. Networking & data persistence (3–4 weeks · Free) URLSession + Codable (iOS) or Retrofit + Room (Android). Build an app that fetches and caches data.

    4. App architecture (3–4 weeks · Mixed) MVVM, Clean Architecture, or TCA (iOS). Architecture determines everything at scale.

    5. Cross-platform with React Native or Flutter (1–2 months · Free) Expo docs + build a production-quality cross-platform app.

    6. App Store deployment & CI/CD (1–2 weeks · Free) Fastlane docs + set up an automated pipeline: test → build → deploy.

    7. On-device ML (2–3 weeks · Free) Apple's CoreML & Create ML tutorials. Build a real app that classifies images or runs speech-to-text.

    • ConstraintLayout (Android)
    • SwiftUI state management
    • Animations & transitions
    • Accessibility on mobile
    • MVVM vs MVC vs TCA
    • Dependency injection
    • Repository pattern
    • Offline-first design
    • App modularization
    • App lifecycle (foreground/background)
    • Push notification flow
    • App Store review process
    • Keychain & secure storage
    • Deep linking architecture
