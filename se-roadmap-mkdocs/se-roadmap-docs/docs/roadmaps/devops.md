# DevOps / Cloud Engineer

Infrastructure, reliability, and CI/CD. Engineers who want to make systems run smoothly.

---

## Foundation

`Linux & Bash scripting` `Networking (TCP/IP, DNS, TLS)` `Git` `Python scripting` `Cloud concepts (IaaS / PaaS / SaaS)`

---

## Core skills

`Docker & container registries` `Terraform / Pulumi (IaC)` `AWS / GCP / Azure` `GitHub Actions / Jenkins` `Prometheus & Grafana` `Ansible`

---

## Advanced

`Kubernetes (EKS / GKE / AKS)` `Service mesh (Istio / Linkerd)` `GitOps (ArgoCD / Flux)` `Secrets management (Vault)` `SRE practices & SLOs` `ELK / OpenTelemetry stack`

---

## Specialization

`Platform engineering` `FinOps & cost optimization` `Multi-cloud strategy` `Chaos engineering` `Zero-downtime deployment` `Compliance automation (SOC2)`

---

!!! tip "Key insight"
    AI/ML infrastructure is a new discipline within DevOps. GPU cluster management, model serving, and LLMOps are fast becoming core skills alongside traditional reliability engineering.
