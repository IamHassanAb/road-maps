# Roadmaps Overview

Each path is organized into four progressive stages. The goal is to be honest about what actually matters at each level — not a laundry list of every tool that exists, but the skills that move the needle.

---

## How to read these roadmaps

**Foundation** skills are non-negotiable prerequisites. If you skip these, you'll be building on sand.

**Core skills** are what you use every day on the job. These are the things hiring managers expect you to know.

**Advanced** skills separate mid-level engineers from seniors. You won't need all of these early, but you should be working toward them.

**Specialization** is what defines staff and principal engineers — deep expertise in one or two areas that teams can't function well without.
