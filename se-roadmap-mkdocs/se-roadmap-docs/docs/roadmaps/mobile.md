# Mobile Engineer

iOS, Android, and cross-platform. Engineers who want to ship apps people carry everywhere.

---

## Foundation

`Swift (iOS) or Kotlin (Android)` `Git` `OOP & design patterns` `UI/UX mobile principles` `Xcode / Android Studio`

---

## Core skills

`UIKit / SwiftUI` `Jetpack Compose` `REST API integration` `Local storage (CoreData, Room)` `Push notifications` `App lifecycle & permissions`

---

## Advanced

`React Native / Flutter (cross-platform)` `Unit & UI testing (XCTest, Espresso)` `CI/CD (Fastlane, Bitrise)` `App Store & Play Store ops` `Performance profiling & memory` `Offline-first architecture`

---

## Specialization

`Custom native modules` `AR (ARKit / ARCore)` `On-device ML (CoreML, TensorFlow Lite)` `Widgets & app extensions` `Deep linking & universal links` `App security & obfuscation`

---

!!! tip "Key insight"
    On-device AI is a major differentiator for mobile apps. Engineers who can run models locally (privacy-first, low-latency) and design AI-native mobile UX are pulling ahead of the field.
