# Data / ML Engineer

AI, analytics, and machine learning. Math-oriented engineers excited about AI.

---

## Foundation

`Python` `SQL` `Statistics & probability` `Linear algebra basics` `Pandas / NumPy` `Jupyter notebooks`

---

## Core skills

`Scikit-learn` `Data pipelines (Airflow, Prefect)` `Spark / Databricks` `Feature engineering` `A/B testing` `Tableau / Power BI / Looker`

---

## Advanced

`PyTorch / TensorFlow` `MLOps (MLflow, Weights & Biases)` `Model serving (FastAPI, TorchServe)` `Kubeflow / Vertex AI` `LLM fine-tuning` `Vector databases (Pinecone, Qdrant)`

---

## Specialization

`Generative AI & RAG pipelines` `Reinforcement learning` `Real-time ML (streaming)` `Model interpretability (SHAP)` `AI safety & alignment` `Cloud ML (SageMaker, Vertex)`

---

!!! tip "Key insight"
    This is the highest-demand path in the AI era. Traditional ML still matters but GenAI skills — fine-tuning, RAG, evals, agents — are now essential in every ML role. The steepest math prerequisite of all paths: statistics and linear algebra underpin everything.
