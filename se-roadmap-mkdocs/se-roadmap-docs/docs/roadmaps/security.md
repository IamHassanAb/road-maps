# Security Engineer

AppSec, pentesting, and AI security. Engineers drawn to adversarial thinking.

---

## Foundation

`Networking (TCP/IP, HTTP, DNS)` `Linux fundamentals` `Python scripting` `Cryptography basics` `OWASP Top 10`

---

## Core skills

`Penetration testing basics` `Burp Suite` `SIEM tools (Splunk, Sentinel)` `Cloud security (IAM, GuardDuty)` `Vulnerability scanning (Nessus, Nuclei)` `Zero trust principles`

---

## Advanced

`Threat modeling (STRIDE / PASTA)` `Red & blue teaming` `Incident response playbooks` `Compliance (SOC2, ISO 27001)` `AppSec in CI/CD (SAST / DAST)` `Container & Kubernetes security`

---

## Specialization

`CVE research & responsible disclosure` `Security architecture review` `Malware analysis & reverse engineering` `Bug bounty programs` `DevSecOps tooling` `Digital forensics`

---

!!! tip "Key insight"
    AI has created an entirely new attack surface: prompt injection, model exfiltration, jailbreaks, and adversarial inputs. AI red teaming is one of the fastest-growing security specializations — and almost no one is trained in it yet.
