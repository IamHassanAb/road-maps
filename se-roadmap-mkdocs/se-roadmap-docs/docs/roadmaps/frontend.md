# Frontend Engineer

UI, browsers, and user experience. Visual thinkers who love fast feedback loops.

---

## Foundation

The baseline every frontend engineer needs before anything else.

`HTML5` `CSS3` `JavaScript ES6+` `Git & GitHub` `Chrome DevTools` `Figma basics`

---

## Core skills

The day-to-day tools you'll use on every frontend job.

`React / Vue / Angular` `TypeScript` `REST APIs` `npm / yarn` `Webpack / Vite` `Responsive design` `SASS / PostCSS`

---

## Advanced

What separates mid-level from senior frontend engineers.

`State management (Redux, Zustand)` `Testing (Jest, Cypress)` `Next.js / Nuxt / Remix` `Performance optimization` `Accessibility (WCAG)` `GraphQL` `CI/CD pipelines`

---

## Specialization

Deep expertise that defines staff and principal-level work.

`Design systems` `Micro-frontends` `WebAssembly` `Progressive Web Apps` `Web animations (GSAP, Framer)` `SSR / ISR patterns`

---

!!! tip "Key insight"
    Frontend engineering is shifting fast. AI can generate layouts and components quickly — your edge is taste, judgment, and the ability to build *around* AI (chat UIs, streaming responses, agentic interfaces). See the [AI Era](../ai-era/frontend.md) page for what this means for your skill priorities.
