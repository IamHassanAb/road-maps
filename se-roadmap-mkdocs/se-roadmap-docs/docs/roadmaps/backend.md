# Backend Engineer

APIs, databases, and distributed systems. Systems thinkers who like working under the hood.

---

## Foundation

`Python / Java / Go / Node.js` `Git & GitHub` `SQL fundamentals` `HTTP & REST concepts` `Data structures & algorithms`

---

## Core skills

`Web frameworks (Django, Spring, Express, Gin)` `PostgreSQL / MySQL` `Authentication (JWT, OAuth2)` `Docker basics` `Redis caching` `Unit testing`

---

## Advanced

`Microservices architecture` `Kafka / RabbitMQ` `Database optimization & indexing` `Kubernetes` `gRPC` `Cloud platforms (AWS / GCP / Azure)` `Distributed tracing`

---

## Specialization

`Distributed systems design` `Event-driven architecture` `CQRS / Event sourcing` `Service mesh (Istio)` `Database sharding` `High-availability patterns`

---

!!! tip "Key insight"
    The most critical new layer for backend engineers is AI integration plumbing — LLM APIs, vector stores, and agent orchestration. Architecture and security judgment matter more as AI handles more boilerplate. See the [AI Era](../ai-era/backend.md) page for the full picture.
