# AI Era — Security Engineer

> **Bottom line:** AI has created an entirely new attack surface: prompt injection, model exfiltration, jailbreaks, and adversarial inputs. AI red teaming is one of the fastest-growing security specializations.

---

!!! success "New AI-era skills"
    - LLM security & prompt injection defense
    - AI red teaming
    - Model extraction & inversion attacks
    - RAG poisoning & data security
    - AI governance & compliance (EU AI Act)
    - Adversarial ML
    - OWASP LLM Top 10

!!! info "Amplified importance"
    - Threat modeling (AI systems)
    - API security
    - Zero trust architecture
    - Supply chain security
    - Incident response (AI incidents)

!!! example "Still essential"
    - Networking & protocols
    - Linux & Bash
    - OWASP Top 10 (web)
    - Burp Suite
    - Cloud security (IAM)

!!! warning "Increasingly automated"

    - Basic SAST/DAST runs
    - Boilerplate security configs
