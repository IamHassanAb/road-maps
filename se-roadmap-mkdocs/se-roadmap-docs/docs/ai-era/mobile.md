# AI Era — Mobile Engineer

> **Bottom line:** On-device AI is a major differentiator for mobile apps. Engineers who can run models locally (privacy-first, low-latency) and design AI-native mobile UX are pulling ahead.

---

!!! success "New AI-era skills"
    - On-device ML (CoreML, TensorFlow Lite, ExecuTorch)
    - AR + AI (ARKit / ARCore + vision models)
    - Local LLMs on device
    - AI-powered UX patterns (smart suggestions, auto-fill)
    - Voice AI integration (Whisper, speech APIs)
    - LLM API integration in mobile apps

!!! info "Amplified importance"
    - Performance & battery optimization
    - Privacy-first architecture
    - Accessibility
    - Native module development
    - App security

!!! example "Still essential"
    - Swift / Kotlin
    - SwiftUI / Jetpack Compose
    - REST API integration
    - Push notifications
    - App Store / Play Store ops

!!! warning "Increasingly automated"
    - Boilerplate list / detail screens
    - Basic form validation
    - Standard navigation patterns
    - Routine API wrappers
