# AI Era — DevOps / Cloud Engineer

> **Bottom line:** AI/ML infra is a new discipline within DevOps. GPU cluster management, model serving, and LLMOps are fast becoming core skills alongside traditional reliability engineering.

---

!!! success "New AI-era skills"
    - GPU & ML infrastructure (CUDA, NCCL)
    - Model serving (vLLM, TorchServe, Triton)
    - LLMOps (experiment tracking, evals)
    - Vector DB ops (Pinecone, Qdrant, Weaviate)
    - AI cost optimization (FinOps for inference)
    - Inference pipeline observability

!!! info "Amplified importance"
    - Platform engineering
    - Security & compliance (SOC2, zero trust)
    - SRE & SLOs
    - GitOps (ArgoCD)
    - Kubernetes (multi-tenant)

!!! example "Still essential"
    - Terraform / Pulumi
    - Docker & K8s basics
    - AWS / GCP / Azure
    - Prometheus & Grafana
    - Linux & Bash

!!! warning "Increasingly automated"
    - Boilerplate IaC templates
    - Standard pipeline configs
    - Routine runbook tasks
    - Basic alerting setup
