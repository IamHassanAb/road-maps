# AI Era — Data / ML Engineer

> **Bottom line:** This is the highest-demand path in the AI era. Traditional ML still matters but GenAI skills — fine-tuning, RAG, evals, agents — are now essential in every ML role.

---

!!! success "New AI-era skills"
    - Fine-tuning & RLHF
    - RAG pipeline design
    - LLM evals & benchmarking
    - AI agents & tool use
    - Prompt engineering (systematic)
    - Multimodal models (vision, audio)
    - Synthetic data generation

!!! info "Amplified importance"
    - MLOps & experiment tracking
    - Python (deeper)
    - Statistics & probability
    - Feature stores
    - Model interpretability (SHAP, LIME)
    - Data quality & pipelines

!!! example "Still essential"
    - PyTorch / TensorFlow
    - SQL & Spark
    - Pandas / NumPy
    - Scikit-learn
    - Jupyter / notebooks

!!! warning "Increasingly automated"
    - Standard EDA templates
    - Basic feature engineering
    - Routine model evaluation scripts
    - Boilerplate training loops
