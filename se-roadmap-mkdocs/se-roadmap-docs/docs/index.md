# Software Engineer Roadmaps

A complete, opinionated guide to software engineering career paths — from first job to staff level, updated for the AI era.

---

## What's inside

=== "Roadmaps"

    The foundational skill stages for each engineering path. Organized into four levels:

    - **Foundation** — baseline skills every engineer on that track needs
    - **Core skills** — the day-to-day tools you'll use on the job
    - **Advanced** — what separates mid-level from senior engineers
    - **Specialization** — deep expertise that defines staff/principal-level work

=== "AI Era Skills"

    How each path changes in the age of AI. For every track you'll find:

    - **New AI-era skills** — things you need that didn't exist 3 years ago
    - **Amplified importance** — skills that matter *more* because of AI
    - **Still essential** — the fundamentals that aren't going anywhere
    - **Increasingly automated** — where AI handles the boilerplate

=== "Deep Dive"

    The full picture for each path:

    - Career levels (Junior → Mid → Senior → Staff/Principal)
    - Tools ecosystem (core vs. useful-to-know)
    - Sequenced learning path with resource recommendations
    - Interview topic breakdown
    - Salary ranges by level (US market, 2024–2025)

---

## The 8 paths

| Path | Focus | Best for |
|------|-------|----------|
| [Frontend](roadmaps/frontend.md) | UI, browsers, user experience | Visual thinkers who love fast feedback loops |
| [Backend](roadmaps/backend.md) | APIs, databases, distributed systems | Systems thinkers who like working under the hood |
| [Full Stack](roadmaps/fullstack.md) | End-to-end product development | Generalists who want to ship complete features |
| [DevOps / Cloud](roadmaps/devops.md) | Infrastructure, reliability, CI/CD | Engineers who want to make systems run smoothly |
| [Data / ML](roadmaps/data-ml.md) | AI, analytics, machine learning | Math-oriented engineers excited about AI |
| [Mobile](roadmaps/mobile.md) | iOS, Android, cross-platform | Engineers who want to ship apps people carry everywhere |
| [Security](roadmaps/security.md) | AppSec, pentesting, AI security | Engineers drawn to adversarial thinking |
| [Eng Manager](roadmaps/eng-manager.md) | People, strategy, org design | Senior engineers ready to lead teams |

---

!!! tip "Cross-cutting advice"
    **Git, a scripting language, and fundamentals** (networking, data structures, HTTP) appear in almost every path. They're the connective tissue regardless of direction. Master these first.

    The jump from **Junior → Mid** is about owning full features. From **Mid → Senior** is about owning systems. From **Senior → Staff** is about owning people and direction.
