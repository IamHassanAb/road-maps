# Deep Dive — Data / ML Engineer

---

## Career levels

=== "Junior (0–2 yrs)"
    **Focus:** Clean data, run exploratory analysis, build simple models, write SQL reports.

    `Python (Pandas, NumPy)` `SQL (intermediate)` `Jupyter notebooks` `Scikit-learn basics` `Data visualization (Matplotlib, Seaborn)` `Statistics fundamentals` `Git`

=== "Mid-level (2–5 yrs)"
    **Focus:** Own ML pipelines, deploy models, build dashboards, drive A/B test analysis.

    `PyTorch or TensorFlow` `Feature engineering` `MLflow or W&B` `Airflow or Prefect pipelines` `Spark basics` `A/B testing (stats)` `Model monitoring` `Looker / Tableau`

=== "Senior (5–10 yrs)"
    **Focus:** Architect ML systems, define experimentation frameworks, lead cross-functional data projects.

    `LLM fine-tuning` `RAG systems` `Distributed training` `MLOps at scale` `Data platform architecture` `Statistical rigor` `Causal inference` `Team leadership`

=== "Staff / Principal (10+ yrs)"
    **Focus:** Define data strategy, lead AI roadmap, advise product and leadership on ML capabilities.

    `GenAI platform strategy` `Foundation model evaluation` `AI governance` `Synthetic data pipelines` `Research collaboration` `Cross-org data standards` `ML platform design`

## Learning path

1. **Python + statistics foundation** *(2–3 months · Mixed)*
   "Python for Data Analysis" (Wes McKinney) + Khan Academy Statistics. Non-negotiable.

2. **Machine learning fundamentals** *(2–3 months · Paid)*
   Andrew Ng's ML Specialization on Coursera. Best structured intro available.

3. **Deep learning with PyTorch** *(2–3 months · Free)*
   "Deep Learning with PyTorch" (official book, free) + fast.ai Practical DL course.

4. **Hugging Face & transformers** *(3–4 weeks · Free)*
   Hugging Face NLP course (free). Build fine-tuned models on real datasets.

5. **MLOps fundamentals** *(1–2 months · Free)*
   Full Stack Deep Learning (free) — covers deployment, monitoring, and MLOps.

6. **LLMs & RAG systems** *(1 month · Paid)*
   DeepLearning.AI short courses (LangChain, RAG, Agents). Build 3 real GenAI projects.

7. **SQL + data warehousing** *(3–4 weeks · Free)*
   Mode Analytics SQL school + dbt fundamentals. Essential for data-adjacent roles.

## Interview topics

=== "ML fundamentals"
    - Bias-variance tradeoff
    - Regularization (L1 vs L2)
    - Cross-validation strategies
    - Class imbalance handling
    - Feature selection methods

=== "Deep learning"
    - Backpropagation intuition
    - Attention mechanism & transformers
    - Overfitting & dropout
    - Batch norm vs layer norm
    - Transfer learning

=== "Statistics & A/B testing"
    - Hypothesis testing & p-values
    - Central limit theorem
    - Bayesian vs frequentist
    - Multiple testing correction
    - Experiment sizing / power

=== "Applied ML"
    - Recommendation system design
    - Fraud detection pipeline
    - RAG system architecture
    - Model monitoring strategies
    - LLM evaluation methods
