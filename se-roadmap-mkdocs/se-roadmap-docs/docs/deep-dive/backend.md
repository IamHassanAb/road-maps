# Deep Dive — Backend Engineer

---

## Career levels

=== "Junior (0–2 yrs)"

    **Focus:** Build CRUD APIs, write SQL queries, work within existing architecture patterns.

    `Python / Node.js / Go basics` `REST API design` `SQL (SELECT, JOIN, INDEX)` `Git & code review` `HTTP fundamentals` `Basic auth (JWT)` `Unit testing`

=== "Mid-level (2–5 yrs)"

    **Focus:** Own services end-to-end, design APIs, improve reliability and performance.

    `System design basics` `PostgreSQL optimization` `Docker & containers` `Redis caching` `Message queues (RabbitMQ)` `OAuth2 / OIDC` `Integration testing` `API versioning`

=== "Senior (5–10 yrs)"

    **Focus:** Architect distributed systems, drive reliability, define patterns used across teams.

    `Microservices patterns` `Kafka / event streaming` `gRPC & Protobuf` `Database sharding & replication` `Distributed tracing (OpenTelemetry)` `Kubernetes` `Security design` `Incident response`

=== "Staff / Principal (10+ yrs)"

    **Focus:** Define system architecture across the org, make build-vs-buy decisions, lead technical strategy.

    `Distributed systems at scale` `Platform abstractions` `Capacity planning` `Cross-team API governance` `AI agent integration design` `Technical writing & RFCs` `Engineering leadership`

## Learning path

1. **Master a backend language** *(2–3 months · Free/Paid)*
   Python with FastAPI or Go are highest ROI in 2025. Build a real REST API project.

2. **SQL & database design** *(1–2 months · Mixed)*
   Mode Analytics SQL tutorial + "Designing Data-Intensive Applications" (Kleppmann).

3. **Distributed systems concepts** *(2–3 months · Paid)*
   "Designing Data-Intensive Applications" is the must-read. Supplement with Martin Fowler blog.

4. **Docker & containerization** *(2–3 weeks · Free)*
   Play With Docker + official docs. Build, ship, and run your app in containers.

5. **Cloud fundamentals (AWS/GCP)** *(1–2 months · Paid)*
   AWS Solutions Architect Associate or GCP Associate Cloud Engineer cert.

6. **System design** *(Ongoing · Paid)*
   ByteByteGo by Alex Xu — covers real interview scenarios and production patterns.

7. **AI integration (LLM APIs)** *(2–3 weeks · Free)*
   Anthropic & OpenAI cookbooks + build a RAG system from scratch.

## Interview topics

=== "Data structures"

    - Arrays, hashmaps, sets
    - Trees & graphs (BFS/DFS)
    - Heaps & priority queues
    - Dynamic programming
    - Sliding window patterns

=== "System design"

    - URL shortener / Pastebin
    - Rate limiter design
    - Design a chat system
    - Distributed caching
    - Database indexing strategies

=== "Databases"

    - ACID vs BASE
    - Normalization & denormalization
    - Query optimization & EXPLAIN
    - Transactions & locking
    - CAP theorem

=== "APIs & architecture"

    - REST vs GraphQL vs gRPC
    - Auth: JWT, OAuth2, OIDC
    - Idempotency & retries
    - Event-driven architecture
    - Microservices tradeoffs
